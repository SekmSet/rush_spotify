#Rush Spotify 

## Config connect to DB

```bash
    Update password in src > Tools > Databse
```

## Install composer

```bash
composer init
composer install
```

### Start server PHP

```bash
php -S localhost:8081 -t public
```

### Start React

```bash
npm install
npm start
```

### Quit server

```bash
CTRL C
```

### Dependencies PHP

* illuminate/database -> eloquent
* illuminate/pagination

### Dependencies JS

* React
* react-js-pagination
* react-router-dom
